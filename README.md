# Hausdiagnose (Next.js)

Marketing / launch site for **Hausdiagnose — Homecheck für gesundes Wohnen** (fokus: **Schweiz**, **de‑CH**, Start im **Raum Zürich**).

## Entwickeln

```bash
pnpm install
pnpm dev
```

Öffne `http://localhost:3000`.

## Qualitätssicherung

```bash
pnpm lint
pnpm test
pnpm build
```

## Deployment

Siehe `DEPLOYMENT.md` (Vercel Root Directory, Env Vars, Preview‑URL / `NEXT_PUBLIC_SITE_URL`, Supabase, optional Upstash Rate Limiting).

## Konfiguration

Siehe `.env.example`.
